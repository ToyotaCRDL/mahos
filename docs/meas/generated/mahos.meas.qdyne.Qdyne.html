mahos.meas.qdyne.Qdyne#

class mahos.meas.qdyne.Qdyne(gconf: dict, name, context=None)#
__init__(gconf: dict, name, context=None)#

Pulse ODMR measurement with Qdyne protocol.

Parameters:

  • pulser.freq (float) – pulse generator frequency (default: 2.0E9)

  • pulser.reduce_start_divisor (int) – the divisor on start of reducing frequency (default: 2) reduce is done first by this value, and then repeated by 10.

  • pulser.split_fraction (int) – fraction factor (F) to split the free period for MW phase modulation (default: 4). the period (T) is split into (T // F, T - T // F) and MW phase is switched at T // F. Thus, larger F results in “quicker start” of the phase modulation (depending on hardware, but its response may be a bit slow).

  • pulser.minimum_block_length (int) – minimum block length in generated blocks (default: 1000)

  • pulser.block_base (int) – block base granularity (default: 4)

  • pulser.start_delay (float) – (sec.) delay time before starting PG output. (default: 0.0)

  • pulser.home_raw_events (str) – (default: “”) Home directory to load RawEvents file.

  • pulser.remove_raw_events (bool) – (default: True) Remove RawEvents file after loading it.
